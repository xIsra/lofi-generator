# Lofi Generator

Infinite procedural lofi to chill you out and focus on your work.

# How it works

The generator is built with React, TypeScript, and Tone.js. It uses a custom song generator to create infinite lofi music.

Using the [Tone.js](https://tonejs.github.io/) library to generate the music.

# How to use

1. Clone the repository
2. Run `pnpm install`
3. Run `pnpm dev`
4. Open [http://localhost:5173](http://localhost:5173) in your browser
